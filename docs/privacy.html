Privacy Policy

This policy explains how Biddr handles basic user information while using the platform.

Information we use

Biddr may use account information such as username, email, avatar, listings, bids, and credits to provide the auction experience.

Local storage

Login information such as access token and username may be stored in the browser to keep users logged in and update the navigation.

How information is used

User information is used to display profiles, listings, bids, and account-related features inside Biddr.

Third-party API

Biddr uses the Noroff API to handle authentication, listings, profiles, bids, and credits.

User control

Users can log out at any time. Logging out removes stored login data from the browser.
