# Biddr

Biddr is a modern online auction marketplace where users can browse listings, place bids, manage auctions, and customize their profile through a clean and responsive user experience.

This project was developed as part of the Semester Project for the Front-End Development program at Noroff.

The application was built using vanilla JavaScript, Tailwind CSS, and the Noroff API v2.
---

## Features

### Authentication
- User registration
- User login/logout
- Protected pages for authenticated users

### Listings
- Browse auction listings
- Search listings
- Featured listings carousel
- Pagination system
- View single listings
- Create listings
- Edit listings
- Delete listings

### Bidding System
- Place bids on active auctions
- View current bids
- View bid history
- Live countdown timer for auctions

### Profile System
- User profile page
- View active and ended listings
- Edit avatar and banner images
- Credits display

### Credits
- Add credits to account
- Credits shown in navigation and profile

### Responsive Design
- Mobile responsive layouts
- Tablet responsive layouts
- Desktop optimized experience
- Hamburger navigation menu for smaller screens
